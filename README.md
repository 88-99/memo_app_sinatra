# memo_app_sinatra(メモアプリ)
## 起動方法
### 1. GitHubからローカルにクローンする
$ git clone git@github.com:88-99/memo_app_sinatra.git 任意のディレクトリ名

### 2. クローンしたディレクトリに移動
$ cd memo_app_sinatra

### 3. bundle installを実行
$ bundle install

### 4. ファイルを実行
$ bundle exec ruby memo_app.rb

### 5. http://localhost:4567/memos にアクセスする
